# **App Name**: MediVoice

## Core Features:

- AI-powered Voice Symptom Capture: Allows users to verbally describe symptoms in multiple Indian regional languages, utilizing advanced speech-to-text and translation to capture medical information.
- AI Clinical Triage & Severity Tool: An AI tool that analyzes captured symptoms and linguistic cues to assess medical urgency and generate a preliminary severity score (Green, Yellow, Red) and guidance.
- Intelligent Navigation & Appointment System: Based on triage results, provides real-time recommendations for navigating to appropriate medical care, such as specialists or emergency services, including booking options.
- User Authentication & Profile Management: Secure sign-up, login, and personal profile management for both patients and healthcare practitioners, ensuring protected access to medical interactions.
- Personalized Triage History Dashboard: A dedicated user interface for reviewing past voice triage sessions, accessing symptom summaries, clinical guidance, and referral details in an organized manner.
- Secure Data Handling & Compliance: Implements military-grade encryption and adheres to HIPAA Tier 3 standards for all voice and clinical data streams, ensuring patient privacy and data integrity.

## Style Guidelines:

- Primary Color: A vibrant, clear cyan-teal (#57f1db) symbolizing technology and medical precision, used for highlights and interactive elements in a dark environment.
- Background Color: A deep, muted blue-black (#0f131d) establishing a sophisticated, low-light setting that provides contrast for the bright primary accents.
- Accent Color: A refreshing, slightly desaturated green (#3fbb7e) providing a subtle, analogous visual contrast to the primary color while maintaining a professional and calm aesthetic.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, tech-inspired character, ensuring bold and impactful headings. Body font: 'Manrope' (sans-serif) for high legibility and contemporary appeal in longer text passages and labels. Monospace font: 'JetBrains Mono' (monospace) for technical information or data snippets.
- Note: currently only Google Fonts are supported.
- Utilize 'Material Symbols Outlined' for clear and universal visual communication, with a mix of filled variations for emphasis where appropriate, aligning with a clean digital interface.
- A dark-themed, minimalist layout featuring grid-based structures, liberal use of 'glassmorphism' for layered UI elements, and abstract geometric or waveform background motifs to convey advanced technology.
- Subtle and purposeful animations including element transitions, soft hover effects, and dynamic waveform visualizations, enhancing user engagement and feedback without causing distraction.
